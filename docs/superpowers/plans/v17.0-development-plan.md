# HJTPX 行为验证系统 v17.0 开发计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use general_purpose_task agent for implementation task-by-task

**Goal:** 完成v17.0开发任务，完善前后端代码，加强人机识别、行为验证、环境检测和代码加密功能

**Architecture:** 采用模块化Go后端+Gin框架，前端使用原生HTML/JS，PostgreSQL+Redis数据存储，支持微服务架构

**Tech Stack:** Go + Gin + GORM, PostgreSQL 16, Redis 7, Bootstrap 5 (bootcdn.cn), Font Awesome 6 (bootcdn.cn)

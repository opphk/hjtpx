Behavior Verification System API

完整的验证码、认证、环境检测与管理API文档

Version 2.0.0

API 概览

Base URL: https://api.example.com/api/v1
开发环境: http://localhost:8080/api/v1

支持的功能

🔐

多种验证码类型

支持滑块验证码、点击验证码、图片验证码、手势验证码

🛡️

环境检测

浏览器指纹采集、自动化工具检测、代理检测

📊

行为分析

轨迹分析、风险评分、设备指纹识别

👥

管理后台

应用管理、日志分析、黑名单管理、风控规则配置

通用响应格式

{ "code": 0, "message": "success", "data": { ... } }

认证方式

API支持两种认证方式：

  • API Key: 在请求头中添加 X-API-Key
  • JWT Token: 在请求头中添加 Authorization: Bearer <token>

快速开始

5分钟快速集成 - 按照以下步骤快速集成验证码服务

1. 安装 SDK

go get github.com/hjtpx/hjtpx/sdk/go

2. 初始化客户端

import ( "github.com/hjtpx/hjtpx/sdk/go" ) client := sdk.NewClient( sdk.WithEndpoint("http://localhost:8080/api/v1"), sdk.WithAPIKey("your-api-key"), )

3. 获取并验证验证码

// 获取滑块验证码 slider, err := client.GetSliderCaptcha(&sdk.SliderCaptchaRequest{ Width: 320, Height: 160, Tolerance: 8, }) // 验证用户滑动 verifyResp, err := client.VerifySliderCaptcha(slider.ChallengeID, "185") if verifyResp.Success { // 验证通过，继续业务逻辑 }

4. 完整示例

package main import ( "fmt" "log" "github.com/hjtpx/hjtpx/sdk/go" ) func main() { client := sdk.NewClient( sdk.WithEndpoint("http://localhost:8080/api/v1"), ) slider, err := client.GetSliderCaptcha(nil) if err != nil { log.Fatal(err) } fmt.Printf("Got captcha: %s\n", slider.ChallengeID) resp, err := client.VerifySliderCaptcha(slider.ChallengeID, "185") if err != nil { log.Fatal(err) } fmt.Printf("Verification: %v\n", resp.Success) }

验证码接口

GET /captcha/slider
▼

获取滑块验证码，返回背景图和滑块图

请求参数

参数名 类型 必填 说明
width integer 否 图片宽度，默认320
height integer 否 图片高度，默认160
tolerance integer 否 容差值，默认8

响应示例

{ "code": 0, "message": "success", "data": { "challenge_id": "slider_abc123", "background_image": "data:image/png;base64,...", "slider_image": "data:image/png;base64,...", "target_x": 185, "puzzle_y": 80, "slider_width": 50, "slider_height": 50 } }
POST /captcha/verify
▼

验证用户输入的验证码答案

请求体

{ "challenge_id": "slider_abc123", "x": 185, "y": 80, "trajectory": [ {"x": 10, "y": 80, "t": 0}, {"x": 50, "y": 80, "t": 100}, {"x": 185, "y": 80, "t": 500} ] }

响应示例

{ "code": 0, "message": "success", "data": { "success": true, "score": 0.85, "message": "Verification passed", "trajectory_result": { "score": 0.85, "passed": true, "reasons": [] } } }
GET /captcha/image
▼

获取图片验证码

请求参数

参数名 类型 必填 说明
type string 否 类型: number/letter/mixed
count integer 否 字符数量，默认4
noise_mode integer 否 噪点模式: 0-无, 1-点, 2-线, 3-曲线
GET /captcha/click
▼

获取点击验证码，需要按正确顺序点击图标

请求参数

参数名 类型 必填 说明
width integer 否 图片宽度，默认400
height integer 否 图片高度，默认300
icon_count integer 否 图标数量，默认9
mode string 否 提示模式: number/arrow
GET /captcha/gesture
▼

获取手势验证码，用户需要按指定顺序连接点位

错误码

错误码 说明 解决方案
400 请求参数错误 检查请求参数格式
401 未授权/认证失败 检查API Key或Token
403 权限不足 联系管理员授权
404 资源不存在 检查Challenge ID
429 请求过于频繁 降低请求频率
500 服务器内部错误 稍后重试

SDK错误类型

var ( ErrNetworkError = errors.New("network error") ErrTimeout = errors.New("request timeout") ErrInvalidResponse = errors.New("invalid response") ErrServerError = errors.New("server error") ErrInvalidParams = errors.New("invalid parameters") ErrVerificationFailed = errors.New("verification failed") ErrRateLimited = errors.New("rate limited") ErrUnauthorized = errors.New("unauthorized") ErrInternalError = errors.New("internal error") )

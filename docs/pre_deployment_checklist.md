# 部署前安全检查清单

## 基础安全

### 身份验证与授权

- [ ] 修改所有默认密码（数据库、Redis、管理员账户）
- [ ] 配置强密码策略（最小长度12字符，包含大小写、数字、特殊字符）
- [ ] 启用多因素认证（MFA）
- [ ] 禁用匿名访问
- [ ] 配置基于角色的访问控制（RBAC）
- [ ] 审计所有管理员账户

### 网络安全

- [ ] 配置 HTTPS（强制 HTTPS 重定向）
- [ ] 配置 SSL/TLS 证书（使用可信 CA 签发的证书）
- [ ] 配置防火墙规则（仅允许必要的端口）
- [ ] 禁用不必要的网络服务
- [ ] 配置 VPC/网络隔离
- [ ] 配置安全组/iptables 规则
- [ ] 启用 DDoS 防护（如 CloudFlare）

### 应用安全

- [ ] 禁用调试模式（GIN_MODE=release）
- [ ] 配置安全响应头
  - [ ] X-Frame-Options: DENY
  - [ ] X-Content-Type-Options: nosniff
  - [ ] X-XSS-Protection: 1; mode=block
  - [ ] Content-Security-Policy
  - [ ] Strict-Transport-Security
- [ ] 配置 CORS 策略
- [ ] 启用 CSRF 防护
- [ ] 配置请求大小限制
- [ ] 配置请求超时

## 数据库安全

### PostgreSQL 安全

- [ ] 配置强数据库密码
- [ ] 启用 SSL 连接（sslmode=require）
- [ ] 配置 pg_hba.conf 访问控制
- [ ] 限制远程访问（仅允许应用服务器 IP）
- [ ] 启用审计日志（pgaudit）
- [ ] 配置连接池限制
- [ ] 定期更新 PostgreSQL 版本

### Redis 安全

- [ ] 配置强密码（requirepass）
- [ ] 禁用危险命令（FLUSHDB, FLUSHALL, CONFIG, KEYS）
- [ ] 限制可访问IP
- [ ] 启用 TLS（如果支持）
- [ ] 配置最大内存限制
- [ ] 启用持久化（AOF）

### 备份安全

- [ ] 配置自动备份
- [ ] 加密备份文件
- [ ] 安全存储备份（异地存储）
- [ ] 定期测试恢复流程
- [ ] 备份验证脚本

## 应用安全

### JWT 配置

- [ ] 配置强 JWT 密钥（至少32字符）
- [ ] 设置合理的过期时间
- [ ] 配置刷新令牌机制
- [ ] 实现令牌黑名单

### 限流配置

- [ ] 启用请求限流
- [ ] 配置每IP限制
- [ ] 配置每用户限制
- [ ] 配置每API Key限制
- [ ] 配置登录失败限制

### 输入验证

- [ ] 验证所有输入参数
- [ ] 防止 SQL 注入（使用参数化查询）
- [ ] 防止 XSS（输出转义）
- [ ] 防止 CSRF
- [ ] 防止路径遍历
- [ ] 防止命令注入

### 日志与审计

- [ ] 配置安全日志
- [ ] 记录所有认证事件
- [ ] 记录所有授权失败
- [ ] 记录敏感操作
- [ ] 配置日志轮转
- [ ] 配置日志告警

## 容器安全

### Docker 安全

- [ ] 使用非 root 用户运行容器
- [ ] 配置只读文件系统
- [ ] 限制容器资源（CPU、内存）
- [ ] 配置网络安全策略
- [ ] 定期更新基础镜像
- [ ] 扫描容器漏洞
- [ ] 配置容器审计

### 镜像安全

- [ ] 使用官方认证镜像
- [ ] 定期更新基础镜像
- [ ] 扫描依赖漏洞
- [ ] 使用最小化镜像（alpine）
- [ ] 配置镜像签名验证

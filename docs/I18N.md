# 国际化 (i18n) 文档

## 概述

系统支持12种语言的完整国际化，包括从左到右 (LTR) 和从右到左 (RTL) 布局支持。

## 支持的语言

| 语言代码 | 语言名称 | 方向 | 本地名称 |
|----------|----------|------|----------|
| en | English | LTR | English |
| zh | Chinese | LTR | 中文 |
| fr | French | LTR | Français |
| de | German | LTR | Deutsch |
| es | Spanish | LTR | Español |
| ru | Russian | LTR | Русский |
| ja | Japanese | LTR | 日本語 |
| ko | Korean | LTR | 한국어 |
| ar | Arabic | RTL | العربية |
| pt | Portuguese | LTR | Português |
| it | Italian | LTR | Italiano |
| nl | Dutch | LTR | Nederlands |

## 前端国际化

### 使用 Hook

```javascript
import { useLanguage } from './hooks/useLanguage';

function MyComponent() {
  const { currentLanguage, changeLanguage, t, isRTL, supportedLanguages } = useLanguage();

  return (
    <div>
      <p>Current language: {currentLanguage}</p>
      <p>Direction: {isRTL ? 'RTL' : 'LTR'}</p>
      <button onClick={() => changeLanguage('zh')}>切换中文</button>
      <p>{t('common.save')}</p>
    </div>
  );
}
```

```javascript
// 翻译: "Hello {name}, you have {count} messages"
t('greeting', { name: 'John', count: 5 })
// 输出: "Hello John, you have 5 messages"
```

## 后端国际化

### I18nService

```javascript
const i18nService = require('./services/i18n_service');

// 初始化
await i18nService.initialize('en');

// 切换语言
await i18nService.setLocale('zh');

// 获取翻译
const message = i18nService.t('auth.loginSuccess');

// 格式化日期
const formattedDate = i18nService.formatDate(new Date(), {
  year: 'numeric',
  month: 'long',
  day: 'numeric'
});

// 格式化数字
const formattedNumber = i18nService.formatNumber(1234567);

// 格式化货币
const formattedCurrency = i18nService.formatCurrency(99.99, 'USD');

// 检查RTL
const isRTL = i18nService.isRTL();

// 获取语言信息
const info = i18nService.getLocaleInfo('ar');
// { name: 'Arabic', nativeName: 'العربية', dir: 'rtl' }
```

## RTL 支持

### CSS

RTL样式在 `styles/rtl.css` 中定义，包括：

- 文本对齐
- 表单元素
- 模态框
- 导航菜单
- 分页
- 图标方向

### Hook

```javascript
import { useDirection } from './hooks/useDirection';

function App() {
  const { direction, setDirection, isRTL } = useDirection();

  return (
    <div className={isRTL ? 'rtl' : 'ltr'}>
      {/* 组件 */}
    </div>
  );
}
```

## 语言切换

### 动态切换

```javascript
// 前端
const { changeLanguage } = useLanguage();
await changeLanguage('ar');

// 后端
await i18nService.setLocale('ar');
```

### 持久化

语言设置保存在 localStorage (前端) 和 session/cookie (后端)。

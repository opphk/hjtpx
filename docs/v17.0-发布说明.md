# HJTPX v17.0 发布说明

**发布日期**: 2026-05-19

## 📋 版本概述

v17.0是HJTPX行为验证系统的重大升级版本，包含了视频验证码、AR验证码、AI模型v3、微服务架构、多租户系统等20个核心功能的开发和优化。本版本进一步巩固了HJTPX在行为验证领域的领先地位，目标是超越极验、易盾、五秒盾等现有产品。

---

## ✨ 新增功能

### 1. 核心验证码功能升级

#### 🎬 视频验证码
- **5种场景类型**: 物体计数、颜色识别、动作识别、图案匹配、序列记忆
- **动态难度调整**: 根据用户表现自动调整验证难度
- **行为数据分析**: 采集用户观看和交互行为特征
- **多语言支持**: 支持中英文提示
- **文件位置**: `backend/internal/service/captcha/video_generator.go`

#### 🥽 AR验证码
- **4种场景类型**: 物体摆放、手势识别、物体旋转、序列动作
- **WebXR集成**: 支持WebXR标准，可在AR设备上使用
- **3D对象交互**: 支持复杂的3D空间交互验证
- **手势识别**: 支持多种手势的识别和验证
- **文件位置**: 
  - 后端: `backend/internal/service/captcha/ar_generator.go`
  - 前端: `frontend/static/js/ar-captcha.js`

#### 🔗 验证码组合验证系统
- **多步验证流程**: 支持多个验证码串联验证
- **智能验证码选择**: 根据用户风险自动选择验证类型
- **动态难度递进**: 验证失败自动提升难度
- **4种验证策略**: all(全部通过)、any(任一通过)、majority(多数通过)、weighted(加权评分)
- **文件位置**: `backend/internal/service/captcha/combo_service.go`

#### 🤖 AI模型升级v3
- **GPT驱动的验证码生成**: 使用AI生成动态验证码内容
- **LSTM特征提取**: 768维特征向量提取
- **深度学习风险评估**: 多层神经网络分析
- **实时行为学习**: 在线学习用户行为模式
- **威胁检测**: 识别30+种异常行为模式
- **文件位置**: `backend/internal/service/captcha/ai_model_v3_enhanced.go`

## 📝 升级指南

### 从v15.0升级到v17.0

1. **数据库迁移**
   ```bash
   # 运行迁移脚本
   psql -h localhost -U verify_user -d verify_db -f scripts/v17_migration.sql
   ```

2. **更新依赖**
   ```bash
   cd backend
   go mod tidy
   go get -u all
   ```

3. **重启服务**
   ```bash
   # 停止旧服务
   pkill hjtpx
   
   # 启动新服务
   ./hjtpx
   ```

4. **验证升级**
   ```bash
   curl http://localhost:8080/health
   ```

## 验证码选型指南

### 验证码类型对比

| 类型 | 安全性 | 易用性 | 适用场景 | 推荐指数 |
|------|--------|--------|----------|----------|
| 滑块验证 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 通用场景、登录注册 | ⭐⭐⭐⭐⭐ |
| 点选验证 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | 高安全需求、注册 | ⭐⭐⭐⭐ |
| 图形验证码 | ⭐⭐⭐ | ⭐⭐⭐⭐ | 低安全场景、评论 | ⭐⭐⭐ |
| 拼图验证 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 高安全场景、支付 | ⭐⭐⭐⭐⭐ |
| 无感验证 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 已注册用户回访 | ⭐⭐⭐⭐ |
| 旋转验证 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 中等安全场景 | ⭐⭐⭐⭐ |
| 手势验证 | ⭐⭐⭐⭐ | ⭐⭐⭐ | 移动端、特殊场景 | ⭐⭐⭐ |

### 场景化选型建议

#### 1. 用户登录
**推荐方案**: 滑块验证 + 风险联动

```yaml
# 配置示例
captcha_type: slider
risk_threshold: 50
max_attempts: 3
enable_behavior_analysis: true
# 高风险用户自动升级为点选验证
```

**原因**:
- 登录是高频操作，滑块验证用户体验好
- 行为分析可识别异常登录
- 3次失败后增强验证强度

#### 2. 用户注册
**推荐方案**: 点选验证或拼图验证

```yaml
captcha_type: click
max_points: 3
difficulty: medium
enable_behavior_analysis: true
```

**原因**:
- 注册需要较强防护，防止机器人批量注册
- 点选/拼图破解难度更高
- 行为分析可检测机器注册模式

#### 3. 找回密码
**推荐方案**: 滑块验证 + 短信/邮箱联动

```yaml
captcha_type: slider
risk_threshold: 40
enable_behavior_analysis: true
```

**原因**:
- 找回密码是敏感操作
- 配合短信/邮箱验证双重保护
- 风险阈值适当降低

#### 4. 评论/发帖
**推荐方案**: 图形验证码或无感验证

```yaml
# 已登录用户使用无感验证
captcha_type: seamless
# 未登录用户使用图形验证码
captcha_type: image
```

**原因**:
- 评论是高频操作，用户体验重要
- 已登录用户可使用无感验证
- 平衡安全性和便捷性

#### 5. 支付/交易
**推荐方案**: 拼图验证 + 行为分析

```yaml
captcha_type: puzzle
difficulty: hard
risk_threshold: 30
enable_behavior_analysis: true
```

**原因**:
- 支付是最高风险场景
- 拼图验证安全性最高
- 低风险阈值确保安全

#### 6. 活动抢购
**推荐方案**: 滑块验证 + 限流

```yaml
captcha_type: slider
risk_threshold: 50
rate_limit:
  per_ip: 100/minute
  per_user: 10/minute
```

**原因**:
- 抢购场景需要快速通过
- 配合限流防止刷单
- 滑块验证速度快

## 安全加固实践

### 1. 签名验证机制

所有API请求应使用签名验证：

```go
// 生成签名
func GenerateSignature(appSecret, timestamp, nonce, requestBody string) string {
    // 1. 拼接参数
    params := fmt.Sprintf("%s%s%s%s", appSecret, timestamp, nonce, requestBody)

    // 2. 计算SHA256
    hash := sha256.Sum256([]byte(params))

    // 3. 返回十六进制字符串
    return hex.EncodeToString(hash[:])
}

// 验证签名
func VerifySignature(appSecret, signature, timestamp, nonce, requestBody string) bool {
    expected := GenerateSignature(appSecret, timestamp, nonce, requestBody)
    return subtle.ConstantTimeCompare([]byte(signature), []byte(expected)) == 1
}
```

### 2. 防重放攻击

```go
// 使用Redis存储nonce
func CheckNonce(nonce string, ttl time.Duration) bool {
    key := fmt.Sprintf("nonce:%s", nonce)
    exists, err := redis.Exists(key).Result()
    if err != nil || exists > 0 {
        return false
    }
    redis.Set(key, "1", ttl)
    return true
}

// 时间戳验证（5分钟窗口）
func CheckTimestamp(timestamp string) bool {
    ts, err := strconv.ParseInt(timestamp, 10, 64)
    if err != nil {
        return false
    }
    now := time.Now().Unix()
    return abs(now-ts) < 300  // 5分钟
}
```

### 4. 风险评分策略

```go
// 风险评分计算
func CalculateRiskScore(ctx *VerificationContext) int {
    score := 0

    // 轨迹异常检测
    if !IsTrajectoryNormal(ctx.Trajectory) {
        score += 30
    }

    // 速度异常检测
    if IsSpeedAnomalous(ctx.Trajectory) {
        score += 20
    }

    // 代理/VPN检测
    if ctx.IsProxy || ctx.IsVPN {
        score += 25
    }

    // 新设备检测
    if ctx.IsNewDevice {
        score += 10
    }

    // 历史行为异常
    if ctx.HasBehaviorAnomaly {
        score += 15
    }

    return score
}

// 风险等级判定
func GetRiskLevel(score int) string {
    switch {
    case score < 30:
        return "low"
    case score < 60:
        return "medium"
    case score < 80:
        return "high"
    default:
        return "critical"
    }
}
```

    }
    return true, reason
}

## 性能优化实践

### 1. 缓存策略

```go
// 多级缓存
type CacheManager struct {
    local  *lru.Cache    // 本地内存缓存
    redis  *redis.Client // Redis缓存
}

// 获取验证码结果
func (cm *CacheManager) GetCaptchaResult(sessionID string) (*CaptchaResult, bool) {
    // 1. 先查本地缓存
    if cm.local != nil {
        if result, ok := cm.local.Get(sessionID); ok {
            return result.(*CaptchaResult), true
        }
    }

    // 2. 再查Redis
    key := fmt.Sprintf("captcha:result:%s", sessionID)
    data, err := cm.redis.Get(key).Bytes()
    if err == nil {
        var result CaptchaResult
        json.Unmarshal(data, &result)
        // 回填本地缓存
        if cm.local != nil {
            cm.local.Add(sessionID, &result)
        }
        return &result, true
    }

    return nil, false
}
```

## 用户体验优化

### 1. 渐进式验证策略

```javascript
// 渐进式验证
class ProgressiveCaptcha {
  constructor(options) {
    this.userHistory = this.loadUserHistory();
    this.currentLevel = this.determineLevel();
  }

  determineLevel() {
    // 信任用户：使用无感验证
    if (this.userHistory.trusted) {
      return 'seamless';
    }

    // 高风险操作：使用强验证
    if (this.isHighRiskAction()) {
      return 'puzzle';
    }

    // 普通用户：使用标准验证
    return 'slider';
  }

  isHighRiskAction() {
    return ['payment', 'password_reset', 'profile_change'].includes(this.actionType);
  }

  upgrade() {
    // 验证失败后升级难度
    const levels = ['slider', 'click', 'puzzle'];
    const currentIndex = levels.indexOf(this.currentLevel);
    if (currentIndex < levels.length - 1) {
      this.currentLevel = levels[currentIndex + 1];
    }
  }
}
```

### 2. 预加载与预验证

```javascript
// 预加载验证码
class CaptchaPreloader {
  constructor() {
    this.preloadOnHover();
    this.preloadOnFocus();
  }

  preloadOnHover() {
    const forms = document.querySelectorAll('form');
    forms.forEach(form => {
      form.addEventListener('mouseenter', () => {
        this.loadCaptcha();
      }, { once: true });
    });
  }

  preloadOnFocus() {
    const inputs = document.querySelectorAll('input[type="text"], input[type="email"]');
    inputs.forEach(input => {
      input.addEventListener('focus', () => {
        this.loadCaptcha();
      }, { once: true });
    });
  }

  loadCaptcha() {
    if (this.captcha) return;
    this.captcha = new HJTPXCaptcha({
      container: '#captcha-container',
      lazyLoad: true
    });
    this.captcha.load();
  }
}
```

### 3. 自动化运维

```bash
#!/bin/bash
# 自动化运维脚本

# 配置常量
BACKUP_DIR="/opt/backups"
LOG_DIR="/var/log/hjtpx"
RETENTION_DAYS=30

# 日志清理
cleanup_logs() {
  echo "清理过期日志..."
  find $LOG_DIR -name "*.log.*" -mtime +$RETENTION_DAYS -delete
  find $LOG_DIR -name "*.log" -mtime +$RETENTION_DAYS -exec truncate -s 0 {} \;
}

# 备份数据库
backup_database() {
  echo "备份数据库..."
  TIMESTAMP=$(date +%Y%m%d_%H%M%S)
  pg_dumpall -U postgres | gzip > $BACKUP_DIR/postgres_$TIMESTAMP.sql.gz
}

# Redis持久化
persist_redis() {
  echo "Redis持久化..."
  redis-cli BGREWRITEAOF
}

# 健康检查
health_check() {
  echo "执行健康检查..."
  curl -s http://localhost:8080/health | jq '.'
}

# 主流程
main() {
  echo "开始运维任务 - $(date)"
  cleanup_logs
  backup_database
  persist_redis
  health_check
  echo "运维任务完成 - $(date)"
}

main "$@"
```

## 安全架构

### 防护层次

```
┌─────────────────────────────────────────────────┐
│                   网络层                          │
│            (防火墙 + WAF + DDoS防护)              │
└─────────────────────────────────────────────────┘
                        │
┌─────────────────────────────────────────────────┐
│                   应用层                          │
│    (签名验证 + CSRF/XSS/SQL防护 + 限流)           │
└─────────────────────────────────────────────────┘
                        │
┌─────────────────────────────────────────────────┐
│                   数据层                          │
│         (加密存储 + 访问控制 + 审计)               │
└─────────────────────────────────────────────────┘
```

### 安全目标

| 目标 | 说明 |
|------|------|
| 机密性 | 保护敏感数据不被未授权访问 |
| 完整性 | 确保数据在传输和存储过程中不被篡改 |
| 可用性 | 确保服务持续稳定运行 |
| 可追溯性 | 记录所有安全相关事件，便于审计 |

## 认证机制

### JWT Token

**Token 结构：**
```json
{
  "header": {
    "alg": "HS256",
    "typ": "JWT"
  },
  "payload": {
    "sub": "user_id",
    "role": "admin",
    "exp": 1716000000,
    "iat": 1715900000
  }
}
```

**配置参数：**
| 参数 | 说明 | 默认值 |
|------|------|--------|
| secret | 签名密钥 | - |
| expire_hours | 过期时间 | 24小时 |
| refresh_hours | 刷新时间 | 168小时(7天) |

**实现位置：**
- `backend/pkg/jwt/jwt.go`

### API Key 认证

**认证流程：**
```
客户端请求
    ↓
携带 API Key 和签名
    ↓
验证签名有效性
    ↓
检查 API Key 权限
    ↓
验证访问频率限制
    ↓
处理请求
```

**签名算法：**
- HMAC-SHA256
- 时间戳验证（防重放）
- 随机数nonce

### 权限控制

**角色说明：**
| 角色 | 说明 |
|------|------|
| admin | 管理员，拥有所有权限 |
| user | 普通用户，基本访问权限 |
| guest | 访客，仅公开接口权限 |

## 安全中间件

### 1. 签名验证

**实现位置：**
- `backend/internal/api/middleware/signature.go`

**验证流程：**
```go
func SignatureValidation() gin.HandlerFunc {
    return func(c *gin.Context) {
        // 1. 获取签名参数
        signature := c.GetHeader("X-Signature")
        timestamp := c.GetHeader("X-Timestamp")
        nonce := c.GetHeader("X-Nonce")

        // 2. 验证时间戳（防重放，5分钟窗口）
        if !validateTimestamp(timestamp) {
            c.AbortWithStatusJSON(401, gin.H{"error": "invalid timestamp"})
            return
        }

        // 3. 验证签名
        expectedSig := generateSignature(c.Request, secret)
        if signature != expectedSig {
            c.AbortWithStatusJSON(401, gin.H{"error": "invalid signature"})
            return
        }

        c.Next()
    }
}
```

## 数据安全

### 传输加密

**HTTPS 强制：**
```nginx
server {
    listen 443 ssl http2;
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers ECDHE-ECDSA-AES128-GCM-SHA256:ECDHE-RSA-AES128-GCM-SHA256;
    ssl_prefer_server_ciphers on;
}
```

**内部通信：**
- 服务间通信使用 mTLS
- API 网关强制 HTTPS

### 存储加密

**敏感字段：**
| 字段 | 加密方式 |
|------|----------|
| 密码 | bcrypt 哈希 |
| API Secret | AES-256-GCM |
| 用户手机号 | AES-256-GCM |
| 身份证号 | AES-256-GCM |

**密码处理：**
```go
func HashPassword(password string) (string, error) {
    bytes, err := bcrypt.GenerateFromPassword([]byte(password), bcrypt.DefaultCost)
    return string(bytes), err
}

func CheckPassword(password, hash string) bool {
    err := bcrypt.CompareHashAndPassword([]byte(hash), []byte(password))
    return err == nil
}
```

### 日志安全

**脱敏处理：**
- 手机号中间4位隐藏
- 身份证号中间8位隐藏
- 邮箱域名隐藏
- 密码完全隐藏

**日志级别：**
| 级别 | 说明 |
|------|------|
| DEBUG | 开发调试 |
| INFO | 正常操作 |
| WARN | 警告事件 |
| ERROR | 错误事件 |
| SECURITY | 安全事件（单独存储） |

## 安全配置

### Nginx 安全配置

```nginx
# 隐藏版本号
server_tokens off;

# 防止点击劫持
add_header X-Frame-Options "SAMEORIGIN" always;

# 防止 MIME 类型嗅探
add_header X-Content-Type-Options "nosniff" always;

# XSS 防护
add_header X-XSS-Protection "1; mode=block" always;

# 强制 HTTPS
add_header Strict-Transport-Security "max-age=31536000; includeSubDomains" always;

# 内容安全策略
add_header Content-Security-Policy "default-src 'self'" always;
```

### CORS 配置

```go
func CORS() gin.HandlerFunc {
    return func(c *gin.Context) {
        c.Header("Access-Control-Allow-Origin", "*")
        c.Header("Access-Control-Allow-Methods", "GET, POST, PUT, DELETE, OPTIONS")
        c.Header("Access-Control-Allow-Headers", "Origin, Content-Type, Authorization")
        c.Header("Access-Control-Max-Age", "86400")

        if c.Request.Method == "OPTIONS" {
            c.AbortWithStatus(204)
            return
        }

        c.Next()
    }
}
```

### 请求验证

```go
func Validation() gin.HandlerFunc {
    return func(c *gin.Context) {
        // 1. 参数长度限制
        if c.Request.ContentLength > 10*1024*1024 { // 10MB
            c.AbortWithStatusJSON(413, gin.H{"error": "request too large"})
            return
        }

        // 2. 请求频率限制
        if !checkRateLimit(c) {
            c.AbortWithStatusJSON(429, gin.H{"error": "rate limit exceeded"})
            return
        }

        // 3. 输入验证
        if !validateInput(c) {
            c.AbortWithStatusJSON(400, gin.H{"error": "invalid input"})
            return
        }

        c.Next()
    }
}
```

### 安全告警

**告警规则：**
| 事件 | 阈值 | 严重级别 |
|------|------|----------|
| 认证失败 | > 100次/分钟 | 高 |
| 限流触发 | > 1000次/分钟 | 中 |
| 异常IP访问 | > 500次/分钟 | 高 |
| SQL注入尝试 | 任何次数 | 严重 |
| XSS尝试 | 任何次数 | 严重 |

### 应急响应

1. **发现入侵：**
   - 隔离受影响服务
   - 保留日志证据
   - 通知安全团队
   - 进行安全审计

2. **数据泄露：**
   - 评估泄露范围
   - 通知受影响用户
   - 升级安全防护
   - 上报监管机构（如需要）

3. **服务攻击：**
   - 启用防护措施
   - 切换备用服务
   - 分析攻击特征
   - 更新防护规则

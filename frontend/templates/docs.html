快速开始

5分钟快速集成

按照以下步骤，您可以在5分钟内将验证码集成到您的网站。

步骤1：注册账号

访问我们的官网完成注册，创建您的应用，获取AppID和AppSecret。

步骤2：引入验证码脚本

在HTML页面中引入验证码SDK脚本。

<script src="https://cdn.example.com/captcha.js"></script>

步骤3：创建验证码容器

在需要验证的位置添加验证码容器。

<div id="captcha-container"></div>

步骤4：初始化验证码

在页面加载完成后初始化验证码组件。

<script>
const captcha = new Captcha('captcha-container', {
  appId: 'YOUR_APP_ID',
  apiBase: '/api/v1',
  type: 'slider',
  onSuccess: function(result) {
    console.log('验证成功:', result);
    // 处理验证成功
  },
  onError: function(error) {
    console.error('验证失败:', error);
    // 处理验证失败
  }
});
</script>
完成！验证码已成功集成到您的网站。

生成滑块验证码

验证码类型

滑块验证

用户通过拖动滑块使拼图块对齐到正确位置。这种方式操作简单，用户体验好，适合大多数场景。

适用场景：
  • 用户登录/注册
  • 表单提交
  • 评论发布

点选验证

用户按照提示依次点击图片中的指定目标。这种方式验证语义理解能力，安全性较高。

适用场景：
  • 高安全性操作
  • 敏感信息修改
  • 支付确认

图形验证

传统的字符或图片识别验证方式。这种方式实现简单，兼容性好，但用户体验相对一般。

适用场景：
  • 兼容性要求高的场景
  • 简单防护需求
  • 无障碍访问

智能验证

后台自动选择最适合的验证方式，无需用户手动切换。根据用户行为和风险评估自动适配。

适用场景：
  • 通用场景
  • 追求最佳用户体验
  • 高并发系统

集成指南

JavaScript集成

<!DOCTYPE html>
<html>
<head>
  <title>验证码集成示例</title>
</head>
<body>
  <form id="login-form">
    <input type="text" name="username" placeholder="用户名" required>
    <input type="password" name="password" placeholder="密码" required>
    <div id="captcha-container"></div>
    <button type="submit">登录</button>
  </form>

  <script src="captcha.js"></script>
  <script>
    let captchaVerified = false;

    const captcha = new Captcha('captcha-container', {
      appId: 'YOUR_APP_ID',
      apiBase: '/api/v1',
      type: 'slider',
      onSuccess: function(result) {
        captchaVerified = true;
        console.log('验证成功:', result);
      },
      onError: function(error) {
        captchaVerified = false;
        console.error('验证失败:', error);
      }
    });

    document.getElementById('login-form').addEventListener('submit', function(e) {
      if (!captchaVerified) {
        e.preventDefault();
        alert('请先完成验证码验证');
        return false;
      }
    });
  </script>
</body>
</html>

React集成

import { useEffect, useRef, useState } from 'react';

function CaptchaComponent({ onVerify }) {
  const containerRef = useRef(null);
  const [verified, setVerified] = useState(false);

  useEffect(() => {
    if (!containerRef.current) return;

    const captcha = new Captcha(containerRef.current, {
      appId: 'YOUR_APP_ID',
      apiBase: '/api/v1',
      type: 'slider',
      onSuccess: function(result) {
        setVerified(true);
        onVerify?.(result);
      }
    });

    return () => {
      captcha.destroy();
    };
  }, [onVerify]);

  return (
    <div>
      <div ref={containerRef}></div>
      {verified && <span>已验证</span>}
    </div>
  );
}

function LoginForm() {
  const handleVerify = (result) => {
    console.log('验证成功:', result);
  };

  const handleSubmit = (e) => {
    e.preventDefault();
    // 处理登录逻辑
  };

  return (
    <form onSubmit={handleSubmit}>
      <input type="text" placeholder="用户名" />
      <input type="password" placeholder="密码" />
      <CaptchaComponent onVerify={handleVerify} />
      <button type="submit">登录</button>
    </form>
  );
}

Vue集成

<template>
  <div>
    <form @submit.prevent="handleSubmit">
      <input v-model="form.username" type="text" placeholder="用户名" />
      <input v-model="form.password" type="password" placeholder="密码" />
      <div ref="captchaRef"></div>
      <button type="submit" :disabled="!verified">登录</button>
    </form>
  </div>
</template>

<script>
import { ref, onMounted, onUnmounted } from 'vue';

export default {
  name: 'LoginForm',
  setup() {
    const captchaRef = ref(null);
    const verified = ref(false);
    const form = ref({ username: '', password: '' });
    let captchaInstance = null;

    onMounted(() => {
      captchaInstance = new Captcha(captchaRef.value, {
        appId: 'YOUR_APP_ID',
        apiBase: '/api/v1',
        type: 'slider',
        onSuccess: (result) => {
          verified.value = true;
          console.log('验证成功:', result);
        }
      });
    });

    onUnmounted(() => {
      captchaInstance?.destroy();
    });

    const handleSubmit = () => {
      if (!verified.value) {
        alert('请先完成验证码验证');
        return;
      }
      // 处理登录逻辑
    };

    return {
      captchaRef,
      verified,
      form,
      handleSubmit
    };
  }
};
</script>

常见问题

注册并登录后，进入控制台，点击“我的应用”，然后点击“创建应用”。系统会自动为您生成AppID和AppSecret。请妥善保管这些信息，不要泄露给他人。

默认情况下，验证码有效期为5分钟（300秒）。您可以在控制台的应用设置中自定义超时时间，最短30秒，最长30分钟。

默认情况下，用户可以无限次重试。但我们建议在连续失败3-5次后要求用户等待一段时间再试，以防止暴力破解。

在初始化Captcha时，通过onSuccess回调处理验证成功的逻辑，通过onError回调处理验证失败的逻辑。验证成功后，您会收到包含session_id的响应，用于后续业务操作的身份验证。

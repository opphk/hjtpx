CaptchaUI 验证码UI组件

验证码状态展示组件，支持加载、成功、失败三种状态，支持中英文切换。

const captcha = new CaptchaUI('container', { type: 'slider', language: 'zh-CN', onSuccess: (data) => console.log('验证成功', data), onError: (error) => console.log('验证失败', error) });

SliderCaptcha 滑块验证组件

完整的滑块验证码交互组件，包含轨迹采集和验证功能。

支持键盘操作：← → 调整位置，Enter 提交

Modal 模态框组件

支持多种尺寸、头部/尾部自定义，键盘导航（ESC关闭）和焦点管理。

Toast 提示组件

自动消失的通知提示，支持多种类型和自定义时长。

无障碍访问支持

所有组件均支持ARIA属性、键盘导航和屏幕阅读器。

  • ARIA角色和标签（role, aria-label, aria-describedby）
  • 键盘导航支持（Tab, Enter, Space, Escape, 方向键）
  • 焦点管理（焦点陷阱，自动聚焦）
  • 屏幕阅读器通知（aria-live, aria-busy）
  • 减少动画偏好支持（prefers-reduced-motion）
  • 高对比度主题支持

主题定制

通过CSS变量轻松定制主题颜色、字体、间距等。

/* 使用CSS变量自定义主题 */ :root { --ui-color-primary: #c9a96e; --ui-color-success: #28a745; --ui-border-radius: 0.5rem; } /* 深色主题 */ [data-ui-theme="dark"] { --ui-bg-primary: #2d2d44; --ui-text-primary: #f8f9fa; }

滑块验证
拖动滑块完成拼图

拖动滑块将拼图块对齐到正确位置。系统实时分析滑动轨迹、速度与加速度，精准区分人类与机器。

点选验证
按序点击指定内容

根据提示文字，依次点击图片中的对应元素。支持汉字、字母、数字多种模式，乱序点选与时序验证双重防护。

点击上方「开始体验」切换至此模式

图形验证码
输入图片中的字符

扭曲变形字符叠加复杂干扰线与噪点，强力抵御 OCR 自动识别。支持自定义字符集与难度等级。

前端集成示例 
// 初始化验证码
const captcha = new Captcha('captcha-container', {
    apiBase: '/api/v1',
    type: 'slider',
    onSuccess: function(result) {
        console.log('验证成功:', result);
    },
    onError: function(error) {
        console.log('验证失败:', error);
    }
});

## 技术架构

### 后端技术栈
- Go 1.21+ / Gin / GORM
- PostgreSQL 12+ / Redis 6+
- gRPC / Protobuf
- WebAssembly

### 前端技术栈
- 原生HTML5/JavaScript
- Bootstrap 5（从bootcdn.cn加载）
- Font Awesome 6（从bootcdn.cn加载）

### AI/ML技术
- TensorFlow Lite
- ONNX Runtime
- 自定义机器学习算法
- 联邦学习框架

### 安全技术
- WebAssembly加密
- 后量子密码学
- 零信任架构
- 区块链存证

## 开发规范

### 代码规范
- 遵循Go代码规范（gofmt）
- 前端代码使用英语注释
- 文档使用中文编写
- 模块化设计

### Git规范
- 使用Conventional Commits
- 分支命名：feat/xxx, fix/xxx, refactor/xxx
- PR描述包含变更内容和测试计划

### 测试规范
- 单元测试覆盖率目标 >98%
- 集成测试覆盖所有API
- E2E测试覆盖核心功能
- 性能测试目标 QPS >15000

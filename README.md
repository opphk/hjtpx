# HJTPX

一个现代化的全栈应用程序

## 快速开始

### 安装
```bash
npm install
```

### 运行
```bash
npm start
```

### 测试
```bash
npm test
```

## 项目结构

├── src/              # 源代码
├── tests/            # 测试文件
├── docs/             # 文档
├── config/           # 配置文件
└── package.json      # 项目配置

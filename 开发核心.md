## API文档

### 基础环境检测API
- `POST /api/v1/detect/check` - 环境检测
- `GET /api/v1/detect/fingerprint` - 指纹查询
- `GET /api/v1/detect/stats` - 统计数据

### 高级环境检测API
- `GET /api/v1/detect/advanced/script` - 获取高级检测脚本
- `POST /api/v1/detect/advanced/submit` - 提交检测数据
- `GET /api/v1/detect/advanced/result` - 获取检测结果
- `POST /api/v1/detect/advanced/browser-engine` - 浏览器引擎分析
- `POST /api/v1/detect/advanced/vm` - 虚拟机检测
- `POST /api/v1/detect/advanced/cloud` - 云环境检测
- `POST /api/v1/detect/advanced/container` - 容器检测
- `POST /api/v1/detect/advanced/headless` - Headless浏览器检测
- `POST /api/v1/detect/advanced/canvas` - Canvas指纹分析
- `POST /api/v1/detect/advanced/webgl` - WebGL指纹分析
- `POST /api/v1/detect/advanced/batch` - 批量检测

### 验证API
- `POST /api/v1/verify` - 验证请求
- `POST /api/v1/verify/validate` - 验证结果校验

### 无感验证API (v5.0)
- `POST /api/v1/trust/evaluate` - 评估设备可信度
- `GET /api/v1/trust/info/:fingerprint` - 获取指纹可信度信息
- `POST /api/v1/trust/verify` - 标记设备为已验证

- `POST /api/v1/trust/risk` - 设置风险评分
- `GET /api/v1/trust/stats` - 获取信任统计
- `POST /api/v1/trust/batch` - 批量评估

### 渐进式验证API (v5.0)
- `POST /api/v1/progressive/start` - 启动渐进式验证
- `POST /api/v1/progressive/challenge` - 完成挑战
- `GET /api/v1/progressive/status/:session_id` - 获取验证状态
- `DELETE /api/v1/progressive/cancel/:session_id` - 取消验证
- `GET /api/v1/progressive/levels` - 获取验证级别
- `POST /api/v1/progressive/result` - 记录验证结果

- `POST /api/v1/whitelist` - 添加白名单
- `DELETE /api/v1/whitelist/:target` - 移除白名单
- `GET /api/v1/whitelist` - 列出白名单
- `GET /api/v1/whitelist/check` - 检查白名单
- `PUT /api/v1/whitelist/:target` - 更新白名单
- `GET /api/v1/whitelist/stats` - 白名单统计
- `POST /api/v1/whitelist/bulk` - 批量添加
- `GET /api/v1/whitelist/export` - 导出白名单

## v4.2 前端安全增强系统 (2026-05-16) ✅

### 新增功能概述

前端加密和安全增强系统是一套完整的Web前端安全解决方案，提供多层次的安全防护机制，包括数据传输加密、代码保护、反调试检测和安全监控等功能。

### 核心模块

#### 1. Web Crypto API 加密模块 (`CryptoAPI`)
- [x] **AES-GCM 对称加密**
  - 支持256位密钥长度的AES加密
  - 随机IV（初始化向量）生成
  - 认证加密，保证数据完整性
  - 自动回退到XOR加密（Web Crypto API不可用时）
  
- [x] **RSA-OAEP 非对称加密**
  - 支持2048/4096位密钥长度
  - 公钥加密、私钥解密
  - SHA-256哈希算法
  - 混合加密方案（大文件分块加密）

- [x] **密钥派生与盐值**
  - PBKDF2密钥派生函数
  - 可配置迭代次数（默认100000）
  - 随机盐值生成
  - 安全的密钥导出/导入

- [x] **哈希与签名**
  - SHA-256哈希计算
  - MD5哈希计算（兼容性）
  - RSA数字签名
  - 签名验证

- [x] **实用工具**
  - Base64编码/解码
  - 十六进制转换
  - 随机字节生成
  - UTF-8编码转换

#### 2. 代码混淆与保护模块 (`CodeObfuscator`)
- [x] **字符串编码加密**

- [x] **变量名混淆**

### 技术实现细节

#### 加密算法实现
- **AES-GCM**: 256位密钥，12字节IV，128位认证标签
- **RSA-OAEP**: 2048位模数，65537公钥指数，SHA-256掩码函数
- **PBKDF2**: SHA-256哈希，可配置迭代次数，默认100000次
- **哈希算法**: SHA-256（256位），MD5（128位，兼容性）

#### 混淆技术
- **字符串编码**: 支持4种编码模式（Unicode/Base64/Hex/RC4）
- **变量混淆**: 随机前缀(_0x) + 字母数字组合
- **死代码**: 随机注入无害代码
- **自毁保护**: 代码完整性动态检测

#### 安全监控机制
- **错误捕获**: window.onerror + unhandledrejection
- **性能观察**: PerformanceObserver API
- **安全检测**: 1秒定时检查DevTools状态
- **可见性**: VisibilityObserver API

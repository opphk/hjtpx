# 开发核心文档 v14.0

## 项目概述

开发一个完整的行为验证系统，前后端均使用Go语言，目标是超越极验、易盾、五秒盾等现有产品。

## 核心特性

- 多种验证方式：滑块验证、点选验证、图形验证码、旋转验证码、手势验证码、拼图验证码、语音验证码、连连看验证码、3D验证码
- 用户端：简洁易用的验证界面
- 管理端：完整的验证数据统计、配置管理、用户行为分析
- 高安全性：AI驱动的行为分析，有效抵御机器攻击
- 自适应难度：根据用户风险动态调整验证难度

## 技术栈

- 后端：Go + Gin + GORM
- 数据库：PostgreSQL + Redis
- 前端：原生HTML/JavaScript（CSS统一从bootcdn.cn加载）
- UI框架：Bootstrap 5（bootcdn.cn）+ Font Awesome 6（bootcdn.cn）
- 监控：Prometheus + Grafana + Loki

## API接口文档

### 用户端验证API

- `POST /api/v1/captcha/slider/generate` - 生成滑块验证码
- `POST /api/v1/captcha/slider/verify` - 验证滑块
- `POST /api/v1/captcha/click/generate` - 生成点选验证码
- `POST /api/v1/captcha/click/verify` - 验证点选
- `GET /api/v1/captcha/image` - 生成图形验证码
- `POST /api/v1/captcha/image/verify` - 验证图形验证码
- `POST /api/v1/captcha/shuffle/click` - 生成乱序点选验证码
- `POST /api/v1/captcha/shuffle/verify` - 验证乱序点选
- `POST /api/v1/captcha/rotate/generate` - 生成旋转验证码
- `POST /api/v1/captcha/rotate/verify` - 验证旋转
- `POST /api/v1/captcha/gesture/generate` - 生成手势验证码
- `POST /api/v1/captcha/gesture/verify` - 验证手势
- `POST /api/v1/captcha/puzzle/generate` - 生成拼图验证码
- `POST /api/v1/captcha/puzzle/verify` - 验证拼图
- `POST /api/v1/captcha/voice/generate` - 生成语音验证码
- `POST /api/v1/captcha/voice/verify` - 验证语音
- `POST /api/v1/captcha/lianliankan/generate` - 生成连连看验证码
- `POST /api/v1/captcha/lianliankan/verify` - 验证连连看
- `POST /api/v1/captcha/3d/generate` - 生成3D验证码

# 行为验证系统 - 开发核心文档 v11.0

## 项目概述

开发一个完整的行为验证系统，前后端均使用Go语言，目标是超越极验、易盾、五秒盾等现有产品。

## 核心特性

- 多种验证方式：滑块验证、点选验证、图形验证码、旋转验证码、手势验证码、拼图验证码、语音验证码、连连看验证码、3D验证码
- 用户端：简洁易用的验证界面
- 管理端：完整的验证数据统计、配置管理、用户行为分析
- 高安全性：AI驱动的行为分析，有效抵御机器攻击
- 自适应难度：根据用户风险动态调整验证难度

## 技术栈

- 后端：Go + Gin + GORM
- 数据库：PostgreSQL + Redis
- 前端：原生HTML/JavaScript（CSS统一从bootcdn.cn加载）
- UI框架：Bootstrap 5（bootcdn.cn）+ Font Awesome 6（bootcdn.cn）
- 监控：Prometheus + Grafana + Loki

## 开发规范

### CSS 使用规范（硬性要求）

- **所有 CSS 必须从 `https://www.bootcdn.cn` 加载，禁止编写任何自定义 CSS 代码**
- 允许使用 `<style>` 内联标签编写少量布局辅助样式（如 flex、margin、padding 等 Bootstrap 无法直接覆盖的场景），但不得编写完整的 UI 样式
- 项目中不得存在任何 `.css` 文件
- 优先使用 Bootstrap 5 内置的 utility classes 实现布局和样式

```

├── backend/                    # 后端服务
│   ├── cmd/                   # 程序入口
│   ├── internal/              # 内部代码
│   │   └── api/
│   │       ├── handler/       # API处理器
│   │       ├── middleware/    # 中间件
```

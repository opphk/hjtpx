## API文档

### 基础环境检测API
- `POST /api/v1/detect/check` - 环境检测
- `GET /api/v1/detect/fingerprint` - 指纹查询
- `GET /api/v1/detect/stats` - 统计数据

### 高级环境检测API
- `GET /api/v1/detect/advanced/script` - 获取高级检测脚本
- `POST /api/v1/detect/advanced/submit` - 提交检测数据
- `GET /api/v1/detect/advanced/result` - 获取检测结果
- `POST /api/v1/detect/advanced/browser-engine` - 浏览器引擎分析
- `POST /api/v1/detect/advanced/vm` - 虚拟机检测
- `POST /api/v1/detect/advanced/cloud` - 云环境检测
- `POST /api/v1/detect/advanced/container` - 容器检测
- `POST /api/v1/detect/advanced/headless` - Headless浏览器检测
- `POST /api/v1/detect/advanced/canvas` - Canvas指纹分析
- `POST /api/v1/detect/advanced/webgl` - WebGL指纹分析
- `POST /api/v1/detect/advanced/batch` - 批量检测

### 验证API
- `POST /api/v1/verify` - 验证请求
- `POST /api/v1/verify/validate` - 验证结果校验

### 管理API
- `POST /api/v1/admin/login` - 管理员登录
- `GET /api/v1/admin/dashboard` - 管理面板数据

## 数据库设计

### 核心表结构

#### fingerprint_records（指纹记录表）
| 字段 | 类型 | 描述 |
|------|------|------|
| id | BIGINT | 主键 |
| visitor_id | VARCHAR(64) | 访客ID |
| fingerprint_hash | VARCHAR(64) | 指纹哈希 |
| risk_score | DECIMAL(5,2) | 风险评分 |
| created_at | TIMESTAMP | 创建时间 |

#### detection_logs（检测日志表）
| 字段 | 类型 | 描述 |
|------|------|------|
| id | BIGINT | 主键 |
| session_id | VARCHAR(64) | 会话ID |
| event_type | VARCHAR(32) | 事件类型 |
| event_data | JSONB | 事件数据 |
| created_at | TIMESTAMP | 创建时间 |

#### risk_rules（风控规则表）
| 字段 | 类型 | 描述 |
|------|------|------|
| id | BIGINT | 主键 |
| rule_name | VARCHAR(64) | 规则名称 |
| rule_type | VARCHAR(32) | 规则类型 |
| condition | JSONB | 触发条件 |
| action | VARCHAR(32) | 执行动作 |
| priority | INT | 优先级 |
| enabled | BOOLEAN | 是否启用 |
| created_at | TIMESTAMP | 创建时间 |

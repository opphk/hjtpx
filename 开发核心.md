# 行为验证系统 - 开发核心文档 v3.0

## 项目概述
开发一个完整的行为验证系统，前后端均使用Go语言，目标是超越极验、易盾、五秒盾等现有产品。

## 核心特性
- 多种验证方式：滑块验证、点选验证、图形验证码、行为分析
- 用户端：简洁易用的验证界面
- 管理端：完整的验证数据统计、配置管理
- 高安全性：AI驱动的行为分析，有效抵御机器攻击

## 技术栈
- 后端：Go + Gin + GORM
- 数据库：PostgreSQL + Redis
- 前端：原生HTML/JavaScript（CSS统一从bootcdn.cn加载）
- UI框架：Bootstrap 5（bootcdn.cn）+ Font Awesome 6（bootcdn.cn）

## 开发规范

### CSS 使用规范（硬性要求）
- **所有 CSS 必须从 `https://www.bootcdn.cn` 加载，禁止编写任何自定义 CSS 代码**
- 允许使用 `<style>` 内联标签编写少量布局辅助样式（如 flex、margin、padding 等 Bootstrap 无法直接覆盖的场景），但不得编写完整的 UI 样式
- 项目中不得存在任何 `.css` 文件
- 优先使用 Bootstrap 5 内置的 utility classes 实现布局和样式

#### 使用示例

**Go SDK**
```go
client := sdk.NewCaptchaClient("app-id", "app-secret", cfg)
slider, _ := client.GenerateSliderCaptcha()
result, _ := client.VerifySliderCaptcha(slider.ChallengeID, "120")
```

**Python SDK**
```python
client = CaptchaClient()
captcha = client.generate_slider_captcha()
result = client.verify_slider_captcha(captcha.challenge_id, "120")
```

**Java SDK**
```java
try (CaptchaClient client = new CaptchaClient()) {
    SliderCaptchaResponse slider = client.generateSliderCaptcha();
    VerifyCaptchaResponse result = client.verifySliderCaptcha(slider.getChallengeId(), "120");
}
```

**JavaScript/TypeScript SDK**
```typescript
const client = new CaptchaClient({ baseUrl: 'http://localhost:8080' });
const slider = await client.generateSliderCaptcha();
const result = await client.verifySliderCaptcha(slider.challenge_id, '120');
```

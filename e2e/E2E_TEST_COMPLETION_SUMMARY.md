  testDir: './tests',

## ⚠️ 重要注意事项

### 1. 环境要求
- ✅ 后端服务必须运行在端口8080
- ✅ PostgreSQL服务必须可用
- ✅ Redis服务必须可用
- ✅ 测试用户账号必须已创建

### 2. 前置条件
在运行E2E测试前，请确保：
```bash
# 1. 启动后端服务
cd /workspace/hjtpx/backend
./hjtpx

# 2. 检查服务健康
curl http://localhost:8080/health

# 3. 安装Playwright浏览器
cd /workspace/hjtpx/e2e
npm run install:browsers
```

### 3. 测试数据清理
部分测试会创建测试数据，建议：
- 定期清理测试数据
- 使用测试数据库环境
- 避免在生产环境运行E2E测试

### 4. 性能考虑
- 并发测试可能对服务器造成压力
- 建议在测试环境运行
- 监控服务器资源使用情况

### 5. 截图管理
- 截图会占用磁盘空间
- 定期清理旧截图（建议每周清理）
- 可以配置CI/CD自动清理

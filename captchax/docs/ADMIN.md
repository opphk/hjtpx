# CaptchaX 管理后台使用手册

## 概述

CaptchaX 管理后台提供可视化的系统管理功能，包括仪表盘、配置管理、白名单/黑名单管理等。

访问地址：`http://your-server:8081/admin/login`

---

## 登录说明

### 首次登录

1. 访问管理后台登录页面
2. 使用默认管理员账号登录：
   - 用户名：`admin`
   - 密码：`admin123`

### 安全建议

- 首次登录后请立即修改默认密码
- 使用强密码（至少8位，包含大小写字母和数字）
- 定期更换密码
- 不要在多个系统使用相同密码

### 登录流程

1. 输入用户名和密码
2. 点击"登录"按钮
3. 验证成功后自动跳转到仪表盘
4. 登录状态有效期为24小时

---

## 仪表盘

登录后默认显示仪表盘页面，展示系统整体运行状态。

### 统计卡片

| 指标 | 说明 |
|------|------|
| 验证码总数 | 统计周期内的验证码生成总数 |
| 验证成功数 | 验证通过的次数 |
| 验证失败数 | 验证失败的次数 |
| 成功率 | 成功次数 / 总次数 × 100% |

### 统计图表

- **验证码趋势图**：展示每日/每周/每月的验证码使用趋势
- **验证类型分布**：滑块、点选、拼图三种验证方式的使用占比
- **验证结果分布**：成功与失败的比例

### 快速操作

| 功能 | 说明 |
|------|------|
| 查看详情 | 点击跳转到详细统计页面 |
| 刷新数据 | 手动刷新统计数据 |
| 导出报告 | 导出统计报表 |

## 配置管理

### 访问路径

仪表盘 → 左侧菜单"系统配置"

### 可配置项

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| max_attempts_per_ip | 10 | 每个IP每小时最大验证次数 |
| block_duration_minutes | 30 | 失败后IP封禁时长（分钟） |
| risk_threshold | 70 | 风险评分阈值 |
| session_timeout_seconds | 300 | 会话超时时间（秒） |
| enable_whitelist | true | 是否启用白名单 |
| enable_blacklist | true | 是否启用黑名单 |

### 修改配置

1. 点击配置项后的"编辑"按钮
2. 输入新的配置值
3. 点击"保存"按钮
4. 配置变更即时生效

### 注意事项

- `max_attempts_per_ip`：设置过小会影响正常用户体验
- `risk_threshold`：数值越低越严格，可能增加误拦率
- 敏感配置修改需要管理员权限

## 白名单管理

### 用途

白名单用于放行可信的 IP 地址，这些 IP 在验证时会跳过风控检查。

### 适用场景

- 内部系统 IP
- 合作方 IP
- 可信用户 IP

### 添加白名单

1. 进入"白名单管理"页面
2. 点击"添加白名单"按钮
3. 填写表单：

| 字段 | 必填 | 说明 |
|------|------|------|
| IP地址 | 是 | 要放行的 IP 地址 |
| 关联域名 | 否 | 关联的域名（可选） |
| 添加原因 | 否 | 备注说明 |

4. 点击"确定"保存

### 查看白名单

1. 进入"白名单管理"页面
2. 可以按 IP 或域名筛选
3. 支持分页查看

### 删除白名单

1. 找到要删除的条目
2. 点击"删除"按钮
3. 确认删除操作

### 批量操作

支持批量导入白名单（按行分隔，每行一个IP）

---

## 黑名单管理

### 用途

黑名单用于封禁恶意 IP，这些 IP 将无法使用验证码服务。

### 适用场景

- 恶意攻击者 IP
- 机器人 IP
- 异常行为 IP

### 添加黑名单

1. 进入"黑名单管理"页面
2. 点击"添加黑名单"按钮
3. 填写表单：

| 字段 | 必填 | 说明 |
|------|------|------|
| IP地址 | 是 | 要封禁的 IP 地址 |
| 封禁原因 | 否 | 封禁原因说明 |
| 过期时间 | 否 | 自动解封时间（留空表示永久封禁） |

4. 点击"确定"保存

### 查看黑名单

1. 进入"黑名单管理"页面
2. 可以按 IP 筛选
3. 可选择"仅显示生效中"
4. 支持分页查看

### 删除黑名单

1. 找到要删除的条目
2. 点击"删除"按钮
3. 确认删除操作

### 自动过期

- 设置了过期时间的黑名单会在到期后自动失效
- 建议定期清理已过期的黑名单条目

## 统计分析

### 访问路径

仪表盘 → 左侧菜单"统计分析"

### 时间范围

支持以下时间范围筛选：

| 选项 | 说明 |
|------|------|
| 24小时 | 最近24小时 |
| 7天 | 最近7天（默认） |
| 30天 | 最近30天 |
| 90天 | 最近90天 |

### 统计维度

| 维度 | 说明 |
|------|------|
| 按验证类型 | 滑块/点选/拼图的分别统计 |
| 按时间段 | 每小时/每天/每周/每月统计 |
| 按应用 | 按不同 app_id 分类统计 |
| 按结果 | 成功/失败分类统计 |

### 导出功能

支持导出 CSV 格式的统计数据

## 用户管理

### 角色说明

| 角色 | 权限 |
|------|------|
| admin | 管理员，拥有所有权限 |
| operator | 操作员，可管理配置和黑白名单 |
| viewer | 查看者，仅可查看数据 |

### 添加管理员

1. 进入"用户管理"页面（仅 admin 角色可见）
2. 点击"添加用户"按钮
3. 填写用户信息
4. 选择用户角色
5. 点击"确定"保存

### 修改密码

1. 点击右上角用户名
2. 选择"修改密码"
3. 输入新密码并确认
4. 点击"保存"

## 常见操作流程

### 封禁恶意IP

1. 在仪表盘发现异常流量
2. 查看验证日志确认恶意IP
3. 进入"黑名单管理"
4. 添加该IP到黑名单
5. 设置合理的过期时间

### 放行内部系统

1. 获取内部系统IP段
2. 进入"白名单管理"
3. 批量添加IP段
4. 验证白名单生效

### 调整风控策略

1. 进入"系统配置"
2. 根据需要调整：
   - 降低阈值提高安全性
   - 提高阈值降低误拦率
3. 观察效果并持续优化

---

## 注意事项

1. **权限管理**：非管理员账号无法修改系统配置
2. **数据安全**：敏感操作会记录审计日志
3. **及时备份**：重要配置变更前建议备份
4. **定期审计**：定期检查白名单/黑名单有效性

💻 快速集成示例
<div id="captcha-container"></div>

<script src="/static/js/captcha.js"></script>
<script>
  const captcha = new SliderCaptcha({
    container: 'captcha-container',
    apiHost: 'http://localhost:8080',
    appId: 'your_app_id',
    onSuccess: (data) => {
      console.log('验证成功', data);
    },
    onFail: (msg) => {
      console.log('验证失败', msg);
    }
  });
</script>
⚙️ 配置选项
参数 类型 默认值 说明
container string 'captcha-container' 验证码容器ID
apiHost string 'http://localhost:8080' 后端API地址
appId string 'default' 应用标识
width number 320 验证码宽度(px)
height number 160 验证码高度(px)
onSuccess function () => {} 验证成功回调
onFail function () => {} 验证失败回调
onError function () => {} 错误处理回调

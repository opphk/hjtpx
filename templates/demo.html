滑块验证码集成示例

使用说明：
  1. 填写用户名和密码
  2. 拖动滑块完成验证
  3. 验证通过后，点击登录按钮

集成方式

方式一：基本集成

最简单的集成方式，只需提供容器ID即可：

new SliderCaptcha({
    container: 'captcha-container'
});
方式二：完整配置

带回调函数和自定义参数：

new SliderCaptcha({
    container: 'captcha-container',
    apiHost: 'http://api.example.com',
    appId: 'my_app',
    width: 320,
    height: 160,
    onSuccess: (data) => {
        console.log('验证成功', data);
    },
    onFail: (msg) => {
        alert('验证失败: ' + msg);
    }
});
方式三：登录表单场景

用于表单提交的验证码验证：

const captcha = new SliderCaptcha({
    container: 'captcha-container',
    onSuccess: () => {
        document.getElementById('submitBtn').disabled = false;
    },
    onFail: () => {
        document.getElementById('submitBtn').disabled = true;
    }
});

document.getElementById('loginForm').addEventListener('submit', (e) => {
    e.preventDefault();
    if (captcha.captchaId) {
        console.log('提交表单');
    }
});
方式四：多实例管理

页面上多个验证码实例：

const captchas = {};

function initCaptcha(id, config) {
    captchas[id] = new SliderCaptcha({
        container: id,
        ...config,
        onSuccess: (data) => {
            console.log(`${id} 验证成功`);
        }
    });
}

initCaptcha('captcha1', { appId: 'app1' });
initCaptcha('captcha2', { appId: 'app2' });

API 文档

HJTPX 验证码系统完整的完整 API 文档

概述

欢迎使用 HJTPX 验证码系统 API。本系统提供多种类型的验证码服务，包括滑动验证码、点击验证码、手势验证码和拼图验证码。

基础信息
  • 基础 URL: /api/v1
  • 数据格式: JSON
  • 字符编码: UTF-8
滑动验证码
获取滑动验证码
GET /api/v1/captcha/slider
响应示例
{
  "success": true,
  "session_id": "abc123",
  "image": "base64_image_data",
  "target_x": 150
}
验证滑动验证码
POST /api/v1/captcha/verify
Content-Type: application/json

{
  "session_id": "abc123",
  "answer": {
    "x": 150
  }
}
响应示例
{
  "success": true,
  "verified": true,
  "message": "验证成功"
}
点击验证码
获取点击验证码
GET /api/v1/captcha/click
验证点击验证码
POST /api/v1/captcha/verify
Content-Type: application/json

{
  "session_id": "abc123",
  "answer": {
    "points": [
      {"x": 100, "y": 50},
      {"x": 200, "y": 100}
    ]
  }
}
手势验证码
获取手势验证码
GET /api/v1/captcha/gesture
验证手势验证码
POST /api/v1/captcha/gesture/verify
Content-Type: application/json

{
  "session_id": "abc123",
  "gesture": "1,2,3,4,5"
}
拼图验证码
获取拼图验证码
GET /api/v1/captcha/jigsaw
验证拼图验证码
POST /api/v1/captcha/jigsaw/verify
Content-Type: application/json

{
  "session_id": "abc123",
  "x": 150
}
健康检查
服务状态检查
GET /health

检查服务是否正常运行，返回 200 状态码表示服务正常。
